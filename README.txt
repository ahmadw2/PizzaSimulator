Run "java PizzaManager" in the console.

The menu will appear, the user must enter the letter 
in the parentheses to perform the following actions.

-----------------------
Welcome to PizzaManager
-----------------------
(A)dd a random pizza
Add a (H)undred random pizzas
(E)at a fraction of a pizza
Sort pizzas by (P)rice
Sort pizzas by (S)ize
Sort pizzas by (C)alories
(B)inary Search pizzas by calories
(Q)uit

Ex. If the user wants to add a random pizza,
they should type "a" and press the enter key
in the console to generate a random pizza.
